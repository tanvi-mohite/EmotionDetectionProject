# 😊 Real-Time Emotion Detection

A real-time facial emotion detection system that identifies human 
expressions (happy, sad, angry, surprised, etc.) using a live webcam feed.

---

## 🔍 About the Project

This project uses **DeepFace** and **OpenCV** to detect and display 
emotions from a webcam in real-time. It was built to learn 
AI, Computer Vision, and Deep Learning with Python.

---

## 🛠️ Technologies Used

- Python 3.x
- DeepFace
- OpenCV (opencv-python)
- TensorFlow / Keras (tf-keras)

---

## ▶️ How to Run

1. Clone this repository
2. Install required libraries:
   pip install -r requirements.txt
3. Run the project:
   python emotion.py

---

## 📸 Output

The webcam opens and detects your emotion live on screen in real-time!
